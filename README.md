# Job Portal Application

## Description
A full-stack Job Portal application with role-based access control for Admin and Candidate users. The system allows candidates to browse and apply for jobs, while admins can manage job postings and view applications.

## Tech Stack
- Java
- Spring Boot
- MySQL
- REST APIs
- HTML, CSS, JavaScript

## Features
### Candidate
- Register and login as a candidate
- View available job listings
- Apply for jobs

### Admin
- Register and login as an admin
- Create, update, and delete job postings
- View all job applications submitted by candidates

### General
- Role-based access control (Admin/Candidate)
- RESTful API integration
- Backend developed using Spring Boot
